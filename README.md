# obabkisan.github.io

Для создания сайта-портфолио была выбрана тема **Material for MkDocs** поскольку она обладает современным адаптивным дизайном, автоматически подстраивается под размер экрана, обеспечивая удобную навигацию как на десктопе, так и на мобильных устройствах.
В текущей версии сайта используется **тёмная тема (Dark Mode)**, тёмный интерфейс снижает нагрузку на глаза при длительной работе с кодом, улучшает читаемость и отражает мой стиль. При необходимости будет добавлен переключатель тем (тёмная/светлая), чтобы каждый посетитель мог выбрать удобный вариант.
