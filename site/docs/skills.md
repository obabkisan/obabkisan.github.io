# Навыки и компетенции

## Python-разработка

### Backend-фреймворки
- **Django** — создание полноценных веб-приложений
- **Flask** — разработка легковесных API и микросервисов
- **FastAPI** — современные высокопроизводительные API

### Машинное обучение
- **NumPy** — научные вычисления и работа с массивами
- **Pandas** — анализ и обработка данных
- **Scikit-learn** — классические алгоритмы ML

### Тестирование
- **unittest** — модульное тестирование
- **pytest** — расширенное тестирование Python-кода
